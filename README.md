Simple plugin for TinyMCE that allows insert images and videos.

## Requirements

* Rails >= 3.1
* TinyMCE4 using the advanced theme

## Setup

### Add the gem to your Gemfile

    gem 'tinymce-rails-perfectmedia'

    # or use git

    gem 'tinymce-rails-perfectmedia', github: 'antonkozhin/tinymce-rails-perfectmedia'
